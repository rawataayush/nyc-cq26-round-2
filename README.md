# 🎓 CollegeIndex

CollegeIndex is a modern college comparison platform that helps students compare engineering colleges based on multiple academic and campus-related parameters. Instead of relying only on rankings, users can personalize weightings and compare institutions according to their own priorities.

---

## 🚀 Tech Stack

- React.js
- Vite
- Tailwind CSS
- React Router DOM
- Remix Icons
